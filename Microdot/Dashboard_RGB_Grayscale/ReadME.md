# Image Dashboard

This is a simple web application built with Microdot, a minimalistic web framework for Python. The application displays an image from a specified URL and allows you to toggle between color and grayscale modes.

## Prerequisites

- Python 3.6 or higher
- Microdot library
- PIL library

## Installation

1. Clone the repository:

   ```
   git clone <repository_url>
   ```

2. Install the required dependencies:

   ```
   pip install microdot Pillow
   ```

## Usage

1. Run the image dashboard application:

   ```
   python image_dashboard.py
   ```

2. Access the application in your web browser at `http://localhost:8008`. You will see the image from the specified URL displayed on the web page.

3. Click the "Toggle Grayscale" button to switch between color and grayscale modes. The image will be dynamically updated based on the selected mode.

## Configuration

- Image URL: Modify the `image_url` variable in the `image_dashboard.py` file to specify the URL of the image you want to display.
